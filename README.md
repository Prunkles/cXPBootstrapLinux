# cXPBootstrapLinux

## Using

```bash
# Go to the directory where bootstrap will be
cd ~/Games/CaveXP/

# Clone repository and change dir to it
git clone https://github.com/Prunkles/cXPBootstrapLinux.git
cd ./cXPBootstrapLinux/

# Make script executable
chmod +x ./cXPBootstrap.sh

# Launch the bootstrap script and play CaveXP!
./cXPBootstrap.sh
```
